# Tracks API & UI — 2026-05-27_10-15 — v0.0.13

> List tracks for the authenticated user. Add optional `sourceId` string attribute (Garmin Connect activity ID).

---

## Context

Existing: `Track` entity has `id`, `user`, `trackName`, `source` (NOT NULL — the system name, e.g. "garmin"),
`uploadTimestamp`, `activityType`, `notes`, `rawPayload`. `TrackRepository.findByUser(user)` already exists.

New `sourceId` = the ID of the activity in the source system (e.g. Garmin Connect activity number). Nullable.
