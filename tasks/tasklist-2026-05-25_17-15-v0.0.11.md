# Minimal React Frontend — 2026-05-25_17-15 — v0.0.11

> Add a minimal React app: Google SSO sign-up for new users, GET /api/me for registered users.
> New pod/service in the `mytracks` k8s namespace. Local dev works without SSO.

---

## Proposed architecture

```
Browser → Traefik ingress
  /mytracks-ui/*  → mytracks-ui Service (nginx, static React build)
  /mytracks/*     → mytracks Service (Spring Boot, existing)
```

Same domain — no CORS issues. React calls `/mytracks/api/...` directly.

### React app flow

```
App loads
  └─ GET /mytracks/api/auth/status
        authenticated: true  → GET /mytracks/api/me → show profile
        authenticated: false → show Google Sign-In button
                                  → POST /mytracks/api/auth/google
                                  → GET /mytracks/api/me → show profile
```

### Local dev flow

```
VITE_GOOGLE_CLIENT_ID unset (or empty)
  → skip Google button
  → GET /mytracks/api/auth/status (mock returns authenticated: true)
  → GET /mytracks/api/me → show profile
Vite proxy: /mytracks → http://localhost:8080
```
